Important

The filter strings provided on this page are sourced from a community-driven Google Sheet. Please ensure you understand the limitations of these filter strings before using them.

Notes

  • You won't find a filter string for all Pokémons, just the ones found in the Google Sheet.
  • Some filter strings maybe too long for iOS, so try one of the shorter strings such as 'Perfect Only' or '100%'.
  • With the exception of 0%, all other filter strings will only include Pokémons with a base attack of 15. i.e. Pokémons with a base attack less than 15 are filtered out regardless of its overall IV percentage.
  • The filter strings are accumulative. i.e. 98%+ includes the Pokémons filtered by both the 'Perfect Only' and '100%' strings.
  • This page is not sync'ed with the Google Sheet. If you notice the Google Sheet has been updated, DM me on Discord, so I can update this page.
